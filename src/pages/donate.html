OUR GOALS

Our plans for the future is to convert this website into a TWA(trusted web activity) and add this website as an app to play store. As TWA's are not allowed in apple store we have to use library like capacitor js to add this website as an app in apple store. If the website helps people, plans are to upgrade database (which costs money) for performance boost.
